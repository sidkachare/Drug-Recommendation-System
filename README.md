# Drug Recommendation System

This project builds a **drug recommendation engine** that suggests similar drugs based on user-input side effects or symptoms using NLP techniques and machine learning. It uses **TF-IDF vectorization** and **KNN** to find drugs with similar side effect profiles.

## Project Structure
